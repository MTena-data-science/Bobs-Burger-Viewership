## Introduction

*Bob’s Burgers* is an animated television show that recently reached its 300th episode in Season 16. It is a show I genuinely enjoy. 

It feels like “chicken soup for the soul,” with a consistent and comforting tone that has helped it last far longer than most animated sitcoms.

That longevity raises interesting data questions. How has viewership changed over time, and do certain seasons perform better than others?

To explore this, I collected episode level viewership data for Seasons 1 through 16 by web scraping episode tables from Wikipedia using Python. 

This analysis focuses on identifying trends in audience viewership and understanding how the show’s performance has evolved across seasons.
